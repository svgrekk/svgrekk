## 🚀 Projects

### [thesis_experiments](https://github.com/svgrekk/thesis_experiments)
Research pipeline for evaluating inference-time hallucination mitigation strategies in LLMs.  
Supports 5 datasets (ASQA, QAMPARI, RAGTruth, TruthfulQA, ELI5), 8 experimental conditions, claim-level verification, and bootstrap confidence intervals.  
`Python` `LLMs` `RAG` `vLLM` `Hugging Face`

### [MedicalDiagAI](https://github.com/svgrekk/MedicalDiagAI)
Hybrid medical assistant prototype combining LLM-based dialogue, symptom extraction, diagnostic classification with XGBoost, and RAG-based follow-up questions.  
`Python` `LLMs` `XGBoost` `RAG` `Gradio`

### [CourierOptimizer](https://github.com/svgrekk/CourierOptimizer)
CLI route planning tool for courier delivery in Oslo. Supports multiple transport modes (car, bike, walk), urgency-weighted scheduling, and optimization by time, cost, or CO₂ emissions.  
`Python` `Optimization` `CLI` `Geospatial`

### [ExploratoryDataAnalysis](https://github.com/svgrekk/ExploratoryDataAnalysis)
