Data Science professional based in Oslo, Norway.

BSc in Applied Data Science (Noroff University College, 2024)

My master's thesis investigated lightweight inference-time hallucination mitigation in large language models — comparing RAG, self-evaluation, conservative prompting, and evidence-constrained prompting across three datasets using a claim-level verification pipeline. The strongest result: evidence-grounded + task-aware self-evaluation reduced hallucination rate from **0.088 to 0.048 on RAGTruth** (−45.5%), with non-overlapping bootstrap confidence intervals.

## 🚀 Projects

### [thesis_experiments](https://github.com/svgrekk/thesis_experiments)
*MSc thesis research code · Grade A*

Evaluation pipeline comparing 8 inference-time hallucination mitigation strategies across 3 datasets (ASQA, QAMPARI, RAGTruth), 300 samples per condition. Implements claim extraction, LLM-based claim verification, metric aggregation, and bootstrap confidence intervals. Best result: RAG + task-aware self-evaluation reduced hallucination rate by 45.5% on RAGTruth with statistically non-overlapping confidence intervals.

`Python` `LLMs` `RAG` `vLLM` `Hugging Face` `NLP`
