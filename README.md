## Projects

### [thesis_experiments](https://github.com/svgrekk/thesis_experiments) · *MSc Thesis, Grade A*
Compared 8 inference-time hallucination mitigation strategies across 3 datasets (ASQA, QAMPARI, RAGTruth) using a shared claim-level verification pipeline. RAG combined with task-aware self-evaluation reduced hallucination rate by **45.5%** on RAGTruth (0.088 → 0.048), the only result with non-overlapping bootstrap confidence intervals across all conditions.

`Python` `LLMs` `RAG` `vLLM` `Hugging Face`
